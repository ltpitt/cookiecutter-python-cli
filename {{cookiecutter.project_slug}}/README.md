## Usage

1. Create a virtual environment.

```bash
$ make venv
```

2. Activate it.

```bash
$ source venv/bin/activate
```

3. Install development dependencies.

```bash
$ make install
```

## Documentation

1. Install documentation-related dependencies.

```bash
$ make docs
```

2. Serve the docs locally.

```bash
$ make serve-docs
```

## Help

For help related to make commands.

```bash
$ make help
```
